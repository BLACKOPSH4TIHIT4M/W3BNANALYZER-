Web3 News Analyzer (Edisi Selenium)Ini adalah alat analisis sentimen berita Web3 yang kuat, dirancang untuk membantu pengguna mengumpulkan, menganalisis, dan memvisualisasikan berita terkait mata uang kripto dari berbagai mesin pencari utama dengan cepat. Proyek ini menggunakan Selenium untuk menggerakkan browser asli guna melakukan pengambilan konten mendalam, secara efektif memecahkan masalah ekstraksi konten halaman web yang dimuat dinamis oleh JavaScript, serta mengintegrasikan analisis sentimen dasar dan fungsi visualisasi kata kunci (hot words).âœ¨ Fitur UtamaPencarian Agregasi Multi-Mesin: Dapat memilih dan menggabungkan hasil dari Bing, Google, Baidu, dan mesin pencari lainnya secara bersamaan, sumber informasi lebih luas.Ekstraksi Konten Mendalam: Menggunakan kerangka kerja Selenium, mampu menggerakkan browser Chrome asli untuk rendering halaman web, secara akurat mengambil teks berita yang dimuat secara dinamis oleh JavaScript, konten lebih lengkap.Targeted Crawling URL: Mendukung input langsung satu atau lebih URL berita oleh pengguna untuk ekstraksi dan analisis konten point-to-point yang presisi.Analisis Sentimen Otomatis: Dilengkapi dengan mesin analisis sentimen berbasis kamus profesional bidang keuangan, secara otomatis menilai kecenderungan "Bullish" (Naik), "Bearish" (Turun), atau "Neutral" (Netral) dari setiap berita.Visualisasi Kata Kunci Keuangan:Word Cloud: Mengagregasi semua konten berita, menyaring kosakata panas yang paling relevan dengan perdagangan keuangan, dan menghasilkan word cloud untuk memahami fokus pasar dengan cepat.Pie Chart Sentimen: Memvisualisasikan hasil analisis sentimen dalam bentuk pie chart, menampilkan suasana opini publik pasar secara keseluruhan secara intuitif.Opsi Konfigurasi Fleksibel: Pengguna dapat dengan bebas mengatur jumlah pencarian, jumlah crawling, rentang waktu berita, dan parameter lainnya.Ringkasan LLM (Opsional): Dapat mengonfigurasi kunci API untuk memanggil Model Bahasa Besar (LLM) guna merangkum dan menyaring semua konten berita secara mendalam.Persistensi Data: Semua data terstruktur yang diambil dan dianalisis (termasuk teks berita dan sentimen) akan disimpan secara otomatis sebagai file CSV untuk memudahkan analisis sekunder.ðŸš€ Mulai CepatPersiapan LingkunganSebelum menjalankan proyek ini, pastikan komputer Anda telah terinstal:Python 3.8 atau versi lebih tinggiGoogle Chrome BrowserLangkah InstalasiKloning atau Unduh ProyekUnduh kode proyek ini ke folder lokal Anda.Instal Pustaka DependensiBuka terminal atau command line, masuk ke direktori root proyek, lalu jalankan perintah berikut untuk menginstal semua pustaka Python yang diperlukan:pip install -r requirements.txt
Konfigurasi ChromeDriverOpsi 1 (Otomatis via APT - Direkomendasikan untuk Kali):sudo apt install chromium-driverOpsi 2 (Manual):Download driver dari situs resmi, ekstrak, lalu arahkan path di config.py ke file tersebut.(Opsional) Konfigurasi API LLMBuka file config.py (BUKAN README ini), dan masukkan API Key Anda di bagian LLM_CONFIG.Menjalankan ProgramSetelah semuanya siap, jalankan perintah berikut di terminal pada direktori root proyek:python main.py
Setelah program dimulai, URL lokal akan ditampilkan (biasanya http://127.0.0.1:7860). Buka URL tersebut di browser Anda untuk mulai menggunakan.ðŸ“– Panduan PenggunaanAlat ini menyediakan dua mode kerja inti:Mode 1: Analisis Agregasi Search EnginePilih Search Engine: Di kotak centang "Search Engine", pilih satu atau lebih mesin pencari yang ingin Anda gunakan.Atur Parameter: Sesuaikan rentang waktu berita, jumlah pencarian, dan jumlah crawling sesuai kebutuhan.Input Kata Kunci: Masukkan topik yang Anda minati di kotak "Input Kata Kunci" (contoh: Dampak Upgrade Ethereum Cancun).Mulai Analisis: Klik tombol "Mulai Analisis".Analisis Cepat: Anda juga bisa langsung mengklik tombol aset kripto preset di bawah (seperti BTC, ETH) untuk analisis satu tombol.Mode 2: Targeted Crawling (URL Spesifik)Ganti Tab: Klik tab "Mode 2: Targeted Crawling".Input URL: Masukkan satu atau lebih URL berita lengkap di kotak teks, satu per baris.Mulai Crawling: Klik tombol "Mulai Crawling Spesifik".Selama program berjalan, area output di sebelah kanan akan menampilkan status tugas, hasil analisis, tabel data, dan grafik visualisasi secara real-time.
